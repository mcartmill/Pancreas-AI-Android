# Pancreas AI
Dexcom CGM live glucose dashboard for Android.

## Setup
1. Open D:\Projects\Pancreas-AI in Android Studio
2. Let Gradle sync finish
3. Run on a device or emulator (API 26+)
4. Open Settings in the app, enter Dexcom Share credentials
5. Tap Test Connection to verify

## Dexcom Share
Enable Sharing inside the Dexcom app: Menu > Share > Enable
US server:      https://share2.dexcom.com
Outside US:     https://shareous1.dexcom.com
