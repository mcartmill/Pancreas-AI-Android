# Pancreas-AI

Open Source insulin/glucose/food tracking app that will make suggestions on insulin amounts and timing.

1/21/2026 - Michael Cartmill

Initial Features:
-Glucose tracking - 2/24/2026 - Changed to use Dexcom OAuth and Share(both working)  Waiting on new Dexcom Dev account and access to recent real data for OAUTH but SHare is recent real data.
-Food tracking
-Insulin tracking - 2/21/2026 - added insulin tracking input - shows on time chart
-Insulin amount suggestions
-Insulin timing suggestions
-Charts



Future Features:
-A1C predictions
-AI integrated for suggestions and pattern recognition
